# 3D Models Directory

Place your downloaded `.glb` or `.gltf` models in this folder.
The RailGuard AI application expects the following exact file names:

1. `station.glb` - The main concourse building or environment.
2. `train.glb` - A realistic train car model.
3. `character.glb` - A VERY low-poly humanoid model for the crowd particles.

If these files are missing, the application will automatically render primitive placeholder shapes (boxes/cylinders) so it doesn't crash.
